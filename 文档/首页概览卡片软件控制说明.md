# 首页概览卡片软件显示控制说明

本文档记录了如何控制面板首页“概览”板块中显示的软件及插件列表。

目前控制首页概览中显示哪些软件，是由前端的 JavaScript 数组硬编码控制的。若要自定义该列表，请遵循以下步骤：

## 1. 修改前端控制列表

打开项目中的前端主逻辑文件：
`web/static/app/index.js`

将你想要显示在概览板块的软件标识名（例如 `'redis'`、`'mariadb'` 等）添加到这个 `plist` 数组中即可。如果不想要某个软件出现在概览中，将其从数组中移除即可。

## 2. 确保后端插件已实现数据统计接口

为了让概览方块能够正确读取并显示**软件的资源统计数值**，你添加到数组中的对应插件，其后端 Python 处理脚本（例如 `plugins/对应插件名/index.py`）的入口调度逻辑中，必须实现 `get_total_statistics` 方法分支。

该接口用于向前端返回软件的统计信息及版本状态。以 `mysql` 和 `gitea` 等插件为例，在它们的 `index.py` 底部调度逻辑中都包含了类似下方的代码：

### 总结

当你需要在首页概览中新增一款软件时，只需：
1. 在 `web/static/app/index.js` 的 `plist` 数组里加上你的插件名。
2. 在 `plugins/插件名/index.py` 的底部功能分发逻辑中添加 `get_total_statistics` 处理分支。
3. 刷新面板首页即可生效。
