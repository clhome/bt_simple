### 🚀 BtSimple V1.0 - 安全加固与仓库迁移正式版

本项目是从 `mdserver-web` 深度 Fork 并重构的轻量级 Linux 面板。本次发布标志着系统已完成全面的安全性审计与性能优化，建议所有用户更新。

#### 🔒 安全增强 (Security Hardening)

- **密码算法升级** ：全面迁移至 `bcrypt` 加希算法，且支持从旧版 MD5 无缝平滑迁移。
- **通信协议锁定** ：强制限制 TLS 1.2+ 协议及高强度加密套件，禁用不安全的旧版 SSL。
- **智能防御机制** ：
- 引入基于 IP 的登录锁定机制，防范暴力破解。
- 强化 CSRF 防护，校验请求来源（Referer/Origin）。
- **注入防护** ：对所有 Shell 命令执行路径进行了 shlex 转义加固，防止命令拼接注入。
- **路径安全** ：增强了文件上传与管理的路径过滤，防止目录遍历攻击。

#### ⚡ 性能与可靠性优化 (Performance & Reliability)

- **数据库连接优化** ：重新配置 SQLite 连接池（10/20 比例），降低资源占用。
- **后台任务重构** ：将高风险的递归重试逻辑改为稳健的循环机制，彻底杜绝 `RecursionError`。
- **响应速度提升** ：移除大量不必要的 `os.system` 调用，改用原生 Python 内置函数处理文件权限，操作延迟显著降低。
- **轮询负载优化** ：优化了后台监控的采样频率，降低闲置时的 CPU 和 I/O 开销。

#### 🌐 仓库迁移说明

- 项目官方地址已正式迁移至：`https://github.com/clhome/bt_simple`
- 更新检查逻辑、部署脚本及 UI 链接已全部同步适配新地址。

#### 🛡️ 防火墙管理深度重构 (Firewall Enhancements)

- **标签化管理** ：重构为“端口防火墙”与“IP 防火墙”双页签模式，提升操作专注度。
- **端口范围交互优化** ：
  - 将单输入框拆分为“开始端口”与“结束端口”，支持输入时自动同步，降低操作频次。
  - 增加端口范围逻辑校验，防止结束端口小于开始端口的违例输入。
- **IP 全局防火墙** ：
  - 新增 IP 级放行/禁止功能，直接作用于系统链首部，优先级高于所有端口规则。
  - **默认拒绝策略**：IP 防火墙新增规则默认为“禁止”状态，强化安全防御意识。
  - **安全闭环** ：增加放行 IP 的风险警示，以及禁止客户端当前 IP 的自毁拦截保护。
- **服务器规则同步 (Server Sync)** ：
  - 新增“服务器同步”按钮，支持一键读取系统底层（Firewalld/UFW/Iptables）的所有规则。
  - **智能入库**：自动识别面板外的手动配置规则并同步至数据库，内置重复规则检测逻辑，确保数据唯一性。
- **高级搜索与布局优化** ：
  - **穿透搜索** ：搜索特定端口时，会自动识别并显示包含该端口的所有范围规则（如搜索 80 可匹配 80:90）。
  - **均衡排版**：优化了规则列表的列宽分配（ID、协议、端口/IP、状态、备注、时间、操作），适配长备注展示，整体视觉更专业。

#### 📂 文件管理功能重构 (File Management Enhancements)

- **多标签页支持 (Tabbed Interface)** ：
  - 引入浏览器风格的标签页功能，支持同时打开多个目录并快速切换。
  - **永久存储**：标签页信息实时保存至浏览器本地存储，关闭浏览器或重启电脑后依然保留。
  - **快捷收藏**：支持一键将当前路径添加为标签，提升高频目录的操作效率。
- **拖拽上传革命 (Drag & Drop)** ：
  - **全屏拖拽**：支持将文件或整个文件夹直接拖入文件列表区域触发上传，极致流畅。
  - **目录结构保留**：深度支持文件夹递归读取，上传后在服务器端完整保留原始目录层级。
  - **预上传确认**：拖入后自动解析文件结构并弹出确认清单，展示项目总数、大小等关键信息，确保上传准确无误。

#### ⏰ 计划任务模块深度优化 (Scheduled Task Enhancements)

- **UI 交互重构** ：
  - **模态框表单**：将繁杂的“添加任务”表单从主视图移至弹窗，界面布局更清爽。
  - **增强工具栏**：新增“添加任务”、“导入”、“导出”按钮组，操作逻辑更符合专业管理习惯。
- **任务导入与导出 (Import & Export)** ：
  - 支持将所有计划任务配置导出为 JSON 文件，便于跨服务器迁移或批量备份。
  - 智能导入机制，支持批量导入并自动过滤重复任务名，防止配置冲突。
- **上次执行时间持久化** ：
  - 任务列表新增“上次执行时间”列，数据实时存储于数据库，告别低效的日志扫描。
  - **自动更新**：无论是手动点击“执行”还是系统自动触发，运行结束后均会自动回传并更新执行时间。
- **智能日期执行策略** ：
  - **日期限制模式**：新增“无、股票开盘日、工作日、节假日”四种执行限制。
  - **节假日 API 集成**：集成 `timor.tech` API，在脚本执行前进行云端日期校验，确保任务仅在预设的日期类型运行。
- **高级搜索与排序** ：
  - **模糊搜索**：支持对任务名称进行实时模糊匹配，快速定位海量任务。
  - **多维点击排序**：支持点击表头（名称、状态、周期、上次执行时间）进行即时排序，并提供绿色的激活视觉反馈。

- **拖拽解析增强** ：
  - 拖拽区域支持实时高亮，支持文件与目录混合拖入，解析速度显著提升。
- **确认清单调优** ：
  - 优化了上传确认弹窗的排版，增加统计数据预览，操作体验更具一致性。
