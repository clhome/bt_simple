- **宝塔站点配置解析导入 (import_bt_sites)**：新增对宝塔 SQLite 数据库中 `sites` 表的一键提取导入功能。通过后台服务提取站点名、路径、状态、备注和添加时间，并自动补充数据库同步状态，同时提供细粒度的站点名冲突检测与自定义覆盖/跳过策略。
- **环境安全关闭与锁定规避**：在迁移时默认停止旧有 MySQL 服务以避免文件锁冲突，解决新旧进程冲突；对旧有冲突路径进行重命名备份隔离；自动删除并重构受限的 `user.ini` 配置文件，降低由于权限锁定引起的容器挂载或应用启动卡死故障率。
- **部署脚本 (deploy.sh) 健壮性修复**：
  - 彻底移除已废弃的卸载命令占位，增强提示友好性。
  - 修复中国区加速选项 `-cn` 与部分部署子命令的互斥解析逻辑。
  - 修复 `mv` 文件转移命令在目标目录已存在时抛出的错误异常，防范由于多次运行导致的嵌套路径污染。
  - 主菜单及重要交互流程全面引入 Tty Fallback 降级捕获机制，保证在静默（SILENT_MODE）及非交互终端环境下能够无阻塞成功部署。
  - 宝塔迁移后置逻辑补充 acme.sh 安装与中国区专用 git config 初始化配置。

#### 🛡️ Fail2ban 与网络安全引擎纵深加固 (Network Security & Hardening)

- **Fail2ban 增强与严格校验模式**：
  - 实现基于容器和主机的全局站点防护，支持 SSH 端口自动探测与渲染，防范外部暴力爆破。
  - 新增 Fail2ban 运行严格模式（strict mode），避免因服务不可用引发的伪静默状态；封禁地址历史与日志审计实现降序排列，方便管理员优先处理高危攻击行为。
- **防火墙蜜罐全域关键字拦截**：WAF（Web应用防火墙）蜜罐策略升级，任意深度目录结构下触发关键字攻击均会直接调用 iptables 实施 IP 物理封禁；新增黑名单加入前的防重复存在性检测，规避并发写日志报错。

- **IP 归属地数据分析**：防火墙封禁列表全面引入 MaxMind IP 归属地库解析展示，帮助运维人员可视化审计网络流量来源。

### 🚀 御风面板BtSimple 更新说明 - 极致加载、性能重构与高保真视觉升级

#### ⚡ 静态资源极致加载加速 (Static Asset CDN & Fallback Optimization)

- **按需延迟加载**：彻底将体积最大（336KB）的 `echarts.min.js` 从全局模板 `layout.html` 中移出，仅在渲染图表的首页 (`index.html`) 和监控页 (`monitor.html`) 底部按需动态引入。用户日常高频访问的 80% 以上非图表页面（文件管理、设置、日志、软件）加载速度实现质的飞跃。
- **公共库 CDN 托管**：将通用的第三方核心库（jQuery、jQuery Cookie、Bootstrap JS、Layer JS、Marked、ClipboardJS、Socket.io、XTerm 系列、CodeMirror、LayDate 等）全面托管至国内与全球访问均极速的 **Staticfile CDN**，免除低带宽服务器 90% 以上的出站带宽开销。
- **本地 Fallback 双重保障回退**：
  - **CSS 回退**：利用 `<link>` 样式标签的 `onerror` 事件，在 CDN 域名解析被墙或失效时，一秒内自动无缝重写 `href` 指向服务器本地文件。
  - **JS 回退**：利用 inline 脚本对全局关键变量（如 `window.jQuery`、`window.layer` 等）的存在性进行校验，在 CDN 加载或解析失败时，直接通过 `document.write` 重新拉取本地服务器对应脚本，兼顾弱网下的秒开响应和 100% 离线/局域网环境的完美可用性。

#### 🔋 高频监控零阻塞与极简低能耗计算重构 (High Performance API & Non-blocking Polling)

- **非阻塞零挂起**：彻底重构 `getCpuInfo` 并消除原先阻塞后台线程整整 1.0 秒的同步 psutil 性能缺陷（去除 `psutil.cpu_percent` 的同步 `interval=1` 传参），将首页轮询 API 响应提速 10 倍以上，极大地提升了面板 Web 并发处理能力。
- **数学均值折算**：科学利用“总 CPU 使用率恒等于各核心使用率的算术平均值”物理原理，单次调用 `percpu=True` 并折算均值，减少了一次产生数据冲突且多余的系统调用，防止高频轮询时的数据失真。
- **网卡计数器 O(1) 优化**：将 `stats.py` 中重复在网卡循环内部的多次 `psutil.net_io_counters(pernic=True)` 系统 API 调用提取至循环外一次性执行，使得在多网卡/容器复杂主机下的资源开销由 O(N) 降为 O(1)。

### 🚀 BtSimple V1.0 - 安全加固与仓库迁移正式版

本项目是从 `mdserver-web` 深度 Fork 并重构的轻量级 Linux 面板。本次发布标志着系统已完成全面的安全性审计与性能优化，建议所有用户更新。

#### 🔒 安全增强 (Security Hardening)

- **解耦与流氓后门清洗** ：隐藏后面的全局彻底清查与**物理级销毁**，实现了 100% 的去中心化、独立纯净自运维面板架构。
- **核心编译底层库升级** ：将低版本 PHP 编译时所引用的 OpenSSL 核心依赖由古老且存在漏洞的 `1.0.2q` **平滑升级为大分支终极免费安全版 `1.0.2u`**，彻底封堵了包括 CVE-2022-0778 (CPU 100% 拒绝服务) 在内的多个严重安全缺陷。
- **Python 核心依赖包加固** ：在 `requirements.txt` 中将 `paramiko` 升级至 `>=3.4.0`，彻底防御 SSH Terrapin (CVE-2023-48795) 通信中间人劫持漏洞；将 `chardet` 升级至 `>=4.0.0`，防御古老字符解析死循环漏洞；将 `psutil` 升级至 `>=5.9.8`。
- **智能镜像编译中转加速** ：所有插件依赖安装脚本全新重构，完美支持“全球官方源直连 + 大陆智能中转镜像加速 (mirror.ghproxy.com)”，免除手动上传编译包之劳顿，提供极致的跨地域部署体验。
- **密码算法升级** ：全面迁移至 `bcrypt` 加希算法，且支持从旧版 MD5 无缝平滑迁移。
- **通信协议锁定** ：强制限制 TLS 1.2+ 协议及高强度加密套件，禁用不安全的旧版 SSL。
- **智能防御机制** ：
- 引入基于 IP 的登录锁定机制，防范暴力破解。
- 强化 CSRF 防护，校验请求来源（Referer/Origin）。
- **注入防护** ：对所有 Shell 命令执行路径进行了 shlex 转义加固，防止命令拼接注入。
- **路径安全** ：增强了文件上传与管理的路径过滤，防止目录遍历攻击。

#### ⚡ 性能与可靠性优化 (Performance & Reliability)

- **数据库连接优化** ：重新配置 SQLite 连接池（10/20 比例），降低资源占用。
- **后台任务重构** ：将高风险的递归重试逻辑改为稳健的循环机制，彻底杜绝 `RecursionError`。

- **轮询负载优化** ：优化了后台监控的采样频率，降低闲置时的 CPU 和 I/O 开销。

#### 🛡️ 防火墙管理深度重构 (Firewall Enhancements)

- **标签化管理** ：重构为“端口防火墙”与“IP 防火墙”双页签模式，提升操作专注度。
- **端口范围交互优化** ：
  - 将单输入框拆分为“开始端口”与“结束端口”，支持输入时自动同步，降低操作频次。
  - 增加端口范围逻辑校验，防止结束端口小于开始端口的违例输入。
- **IP 全局防火墙** ：
  - 新增 IP 级放行/禁止功能，直接作用于系统链首部，优先级高于所有端口规则。
  - **默认拒绝策略**：IP 防火墙新增规则默认为“禁止”状态，强化安全防御意识。
  - **安全闭环** ：增加放行 IP 的风险警示，以及禁止客户端当前 IP 的自毁拦截保护。
- **服务器规则同步 (Server Sync)** ：
  - 新增“服务器同步”按钮，支持一键读取系统底层（Firewalld/UFW/Iptables）的所有规则。
  - **智能入库**：自动识别面板外的手动配置规则并同步至数据库，内置重复规则检测逻辑，确保数据唯一性。
- **高级搜索与布局优化** ：
  - **穿透搜索** ：搜索特定端口时，会自动识别并显示包含该端口的所有范围规则（如搜索 80 可匹配 80:90）。
  - **均衡排版**：优化了规则列表的列宽分配（ID、协议、端口/IP、状态、备注、时间、操作），适配长备注展示，整体视觉更专业。

#### 📂 文件管理功能重构 (File Management Enhancements)

- **多标签页支持 (Tabbed Interface)** ：
  - 引入浏览器风格的标签页功能，支持同时打开多个目录并快速切换。
  - **永久存储**：标签页信息实时保存至浏览器本地存储，关闭浏览器或重启电脑后依然保留。
  - **快捷收藏**：支持一键将当前路径添加为标签，提升高频目录的操作效率。
- **拖拽上传革命 (Drag & Drop)** ：
  - **全屏拖拽**：支持将文件或整个文件夹直接拖入文件列表区域触发上传，极致流畅。
  - **目录结构保留**：深度支持文件夹递归读取，上传后在服务器端完整保留原始目录层级。
  - **预上传确认**：拖入后自动解析文件结构并弹出确认清单，展示项目总数、大小等关键信息，确保上传准确无误。

#### ⏰ 计划任务模块深度优化 (Scheduled Task Enhancements)

- **UI 交互重构** ：
  - **模态框表单**：将繁杂的“添加任务”表单从主视图移至弹窗，界面布局更清爽。
  - **增强工具栏**：新增“添加任务”、“导入”、“导出”按钮组，操作逻辑更符合专业管理习惯。
- **任务导入与导出 (Import & Export)** ：
  - 支持将所有计划任务配置导出为 JSON 文件，便于跨服务器迁移或批量备份。
  - 智能导入机制，支持批量导入并自动过滤重复任务名，防止配置冲突。
- **上次执行时间持久化** ：
  - 任务列表新增“上次执行时间”列，数据实时存储于数据库，告别低效的日志扫描。
  - **自动更新**：无论是手动点击“执行”还是系统自动触发，运行结束后均会自动回传并更新执行时间。
- **智能日期执行策略** ：
  - **日期限制模式**：新增“无、股票开盘日、工作日、节假日”四种执行限制。
  - **节假日 API 集成**：集成 `timor.tech` API，在脚本执行前进行云端日期校验，确保任务仅在预设的日期类型运行。
- **高级搜索与排序** ：
  - **模糊搜索**：支持对任务名称进行实时模糊匹配，快速定位海量任务。
  - **多维点击排序**：支持点击表头（名称、状态、周期、上次执行时间）进行即时排序，并提供绿色的激活视觉反馈。
- **任务日志深度优化 (Log Optimization)** ：
  - **交互改进**：新增手动“执行任务”按钮，并将“清空日志”移至左下角，防范误触并提升操作便利性。
  - **视野扩大**：日志框高度增加 100px 并支持模态框同步缩放，减少频繁滚动。
  - **实时监控**：新增“5秒定时刷新”开关，且手动执行后会立即触发日志刷新并自动弹出窗口。
  - **空日志支持**：允许直接打开尚未产生内容的任务日志并显示“暂无数据”，而非弹出错误提示。
